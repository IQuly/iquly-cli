# IQuly CLI

> **Status:** Coming soon — public release is being prepared.

Command-line tool for package authors: authenticate, scaffold from the blueprint, deploy versions, and chat with a workspace from the terminal.

## Planned commands

- `iquly login` / `iquly whoami` — authenticate and show the active account
- `iquly init` — scaffold a new package from the [agent blueprint](https://github.com/IQuly/iquly-agent-blueprint)
- `iquly deploy` — publish package versions to IQuly
- `iquly chat` — talk to a workspace from the terminal

Guides and reference will live in [iquly-docs](https://github.com/IQuly/iquly-docs).
