# IQuly CLI

> **Status:** Public release is being prepared.

The CLI will provide:

- `iquly login` / `iquly whoami`
- `iquly init` — scaffold from the [agent blueprint](https://github.com/IQuly/iquly-agent-blueprint)
- `iquly deploy` — publish package versions
- `iquly chat` — talk to a workspace from the terminal

Documentation will ship with [iquly-docs](https://github.com/IQuly/iquly-docs).

**Not ready for external use yet.** Watch this repo for the first public release.
